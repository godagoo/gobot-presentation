01 // OVERVIEW

What is GoBot?

A Telegram-based AI interface that brings the power of Claude to your pocket. GoBot remembers context, handles complex requests, and asks for confirmation before taking important actions.

Telegram Native

Works where you already communicate. No new apps to learn.

Context Aware

Remembers conversations, goals, and your preferences across sessions.

03 // SYSTEM

How It Works

GoBot uses a three-tier architecture for reliability and flexibility.

  • VPS Gateway — Receives messages, routes to processor
  • Mac Relay — Local processing with full MCP tool access
  • VPS Processor — Cloud processing with Agent SDK
Telegram → VPS Gateway ↓ ├─→ Mac Relay (local) │ └─→ Claude Code + MCP │ └─→ VPS Processor (cloud) └─→ Agent SDK

04 // DEPLOYMENT

05 // INTELLIGENCE

Smart Routing

GoBot automatically routes messages to the best processor based on complexity and available resources.

  • Simple questions → Fast Haiku API
  • Complex tasks → Agent SDK with tools
  • Force VPS mode with /vps prefix

Complexity Detection

Keywords like "research", "email", "calendar" trigger tool-enabled processing. Everything else uses the fast direct API for instant responses.
